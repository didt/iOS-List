===== Description =====

A jQuery UI Widget for creating an iOS style categorized list from nested lists

===== Usage =====

See demo.html for complete usage.

$('#list').iOSList();

<ul id="list">
    <li>
        <div>Category One</div>
        <ul>
            <li>List Item 1</li>
            <li>List Item 2</li>
            <li>List Item 3</li>
            <li>List Item 4</li>
            <li>List Item 5</li>
        </ul>
    </li>
    
    <li>
        <div>Category Two</div>
        <ul>
            <li>List Item 1</li>
            <li>List Item 2</li>
            <li>List Item 3</li>
            <li>List Item 4</li>
            <li>List Item 5</li>
        </ul>
    </li>
</ul>
